# anoma-apps

This is the `anoma-apps` monorepo. Within it, you will find the following packages:

```bash
anoma-apps/
├── packages/
│   ├── anoma-wallet    # Main wallet React App
│   ├── anoma-lib       # Rust lib for generating WASM
│   └── key-management  # Key management library
```
